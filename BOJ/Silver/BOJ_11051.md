#### 문제

자연수 N과 정수 K가 주어졌을 때 이항 계수 (NK)를 10,007로 나눈 나머지를 구하는 프로그램을 작성하시오.

#### 입력

첫째 줄에 N과 K가 주어진다. (1 ≤ N ≤ 1,000, 0 ≤ K ≤ N)

#### 출력

(NK)를 10,007로 나눈 나머지를 출력한다.

<br/>

## 풀이

#### (1) 2차원 배열

```c++
#include <stdio.h>
int d[1111][1111] = {0, };

int main() {
    int n, k;
    scanf("%d %d", &n, &k);

    for(int i = 1; i <= n; i++) {
        for(int j = 0; j <= i; j++) {
            if(j == 0) {
                d[i][j] = 1;
            } else if(j == 1) {
                d[i][j] = i;
            } else {
                d[i][j] = (d[i-1][j-1] + d[i-1][j]) % 10007;
            }
        }
    }

    printf("%d", d[n][k]);

    return 0;
}
```

<br/>

#### (2) 1차원 배열

```c++
#include <stdio.h>
int main() {
    int n, k, d[1001];
    scanf("%d %d", &n, &k);

    d[0] = 1, d[1] = 1;
    for(int i = 2; i <= n; i++) {
        for(int j = i; j > 0; j--) {
            d[j] = (d[j] + d[j-1]) % 10007;
        }
    }

    printf("%d", d[k]);
    return 0;
}
```
